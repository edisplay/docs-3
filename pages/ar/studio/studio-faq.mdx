---
title: الأسئلة الشائعة حول Subgraph Studio
---

### 1. كيف يمكنني إنشاء مفتاح API؟

في Subgraph Studio ، يمكنك إنشاء API Keys وذلك حسب الحاجة وإضافة إعدادات الأمان لكل منها.

### 2. هل يمكنني إنشاء أكثر من API Keys؟

A: نعم يمكنك إنشاء أكثر من API Keys وذلك لاستخدامها في مشاريع مختلفة. تحقق من الرابط [هنا](https://thegraph.com/studio/apikeys/).

### 3. كيف أقوم بتقييد نطاق مفتاح API؟

بعد إنشاء مفتاح API ، يمكنك في قسم Security تعريف المجالات التي يمكنها الاستعلام عن مفتاح API محدد.

### 4. كيف يمكنني العثور على URL الاستعلام لـ subgraphs إذا لم أكن مطورا لـ subgraph الذي أريد استخدامه؟

يمكنك العثور على URL الاستعلام لكل subgraph في قسم Subgraph Details في The Graph Explorer. عند النقر فوق الزر "Query" ، سيتم توجيهك إلى جزء حيث يمكنك عرض URL الاستعلام لـ subgraph الذي تهتم به. ويمكنك بعد ذلك استبدال `<api_key>` بمفتاح API الذي ترغب في الاستفادة منه في Subgraph Studio.

تذكر أنه يمكنك إنشاء مفتاح API والاستعلام عن أي subgraph منشور على الشبكة ، حتى إذا قمت ببناء subgraph بنفسك. حيث أن هذه الاستعلامات عبر مفتاح API الجديد ، هي استعلامات مدفوعة مثل أي استعلامات أخرى على الشبكة.
